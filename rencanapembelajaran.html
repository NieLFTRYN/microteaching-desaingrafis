Rencana Pembelajaran

Mata Pelajaran: Desain Grafis (Fase F) • Microteaching 15 Menit

Jenjang Pendidikan : SMK
Kelas/Fase : XI / Fase F
Materi : Pengenalan Software Komputasi Vektor (CorelDRAW)
Tema : Eksplorasi Ruang Kerja & Tools CorelDRAW

Tujuan Pembelajaran

Siswa mampu mengidentifikasi komponen user interface (UI) utama CorelDRAW serta mempraktikkan fungsi komponen dasar (Pick Tool, Shape Tool, Rectangle Tool) secara kolaboratif melalui kerja kelompok skala kecil.

Strategi Pembelajaran

Pendekatan Project-Based Learning singkat dengan metode Demonstrasi Interaktif yang dipadukan dengan diskusi kelompok skala kecil (Peer-Pairing / Kelompok Berpasangan) untuk memecahkan studi kasus manipulasi objek vektor.

Alat & Media

  • Buku Panduan Praktis CorelDRAW Fase F.
  • Lembar Kerja Kelompok (LKPD Mini Grafis).
  • PC/Laptop terinstal software CorelDRAW.
  • Proyektor Media Presentasi Ruang Kerja Software.
